FAQ

자주 묻는 질문

궁금한 점이 더 있으시면 언제든지 도입 문의로 연락 주세요.

세븐닷은 어떤 학원에 적합한가요? ＋
국·영·수를 다루는 초·중등 학원에 가장 적합합니다. 학원 규모는 1인 공부방부터 100명 이상의 중대형 학원까지 모두 지원합니다. 학원장이 직접 콘텐츠를 설계하고, 학생들에게 매일 7문제씩 자동 배포하는 일일학습지 시스템입니다.
학생들이 별도 앱을 설치해야 하나요? ＋
학생도, 학원장도 별도 프로그램 설치 없이 크롬 브라우저로 바로 시작할 수 있습니다. PC·노트북·태블릿 어디서나 동일한 화면으로 사용 가능합니다.
브릭맵이 무엇인가요? ＋
학생이 푼 단원을 블록(브릭) 형태로 시각화한 학습 지도입니다. 완벽하게 푼 단원은 골드 브릭, 일부 정답은 실버, 도전 중인 단원은 브론즈로 표시되며, 학생은 골드 브릭을 받을 때까지 같은 단원을 반복할 수 있고 언제든 복습이 가능합니다.
학원이 직접 만든 콘텐츠를 올릴 수 있나요? ＋
네, 가능합니다. 학원이 보유한 데이터를 주시면 세븐닷이 학생에게 자동으로 배포해드립니다 (현재 BETA 단계). 자세한 형식과 절차는 도입 문의로 연락 주세요.
학부모에게는 어떤 알림이 가나요? ＋
매주 월요일에 학생의 일주일 학습 결과가 카카오톡으로 자동 발송됩니다. 한 달 단위 리포트도 월초에 종합 발송됩니다.
도입 문의는 어떻게 하나요? ＋
페이지 하단 "도입 문의하기" 버튼 또는 요금제의 "도입 문의"를 통해 연락 주시면 됩니다. 담당자가 1영업일 이내 연락드리고, 학원 규모·과목 구성에 맞춘 맞춤 컨설팅과 데모를 제공합니다.
요금은 어떻게 결제되나요? ＋
월 단위 자동 결제 또는 연 단위 결제 중 선택 가능합니다. 연 단위 결제 시 추가 할인이 적용되며, 도입 학원의 학생 수 변동에 따라 플랜을 자유롭게 변경할 수 있습니다.
